# ADI Internship 2026

Welcome to the class repository! Here's how to upload your assignments using the website (no terminal needed).

## 📁 How to Upload Files

1. Go to the repository: https://github.com/yossef-helal/ADI-Internship-2026
2. Click the branch dropdown at the top left (currently showing "main")
3. Select **your name** from the list
4. Click **"Add file"** → **"Create new file"** or **"Upload files"**
5. Upload or create your files
6. Scroll down and click **"Commit changes"**
7. Done! Your files are saved.

## 🔍 View Your Work

After uploading, go back to the main page and select your branch to see your files.

## ⚠️ Important

- Only work on **your own branch**
- Don't upload to the "main" branch
- You can upload any file type (PDF, Word, code, images, etc.)
